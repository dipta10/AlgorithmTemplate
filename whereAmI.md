- first prime divisor?

  ```cpp
    /*
     * find the smallest prime divisor!
     * The first divisor that I'll find must be a prime number.
     * else the number itself is a prime number, so return it.
     */
    ll div(ll n) {
        for (ll i = 2; i*i <= n; ++i) if (n % i == 0) return i;
        return n;
    }
  ```
